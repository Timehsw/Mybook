# 安装brew cask

```
brew cask
```

## 预览markdown

```
brew cask install qlmarkdown
```

## BetterZipQL

BetterZipQL 这款插件可以让用户直接使用 QuickLook 查看 Zip 压缩文件的信息以及文件目录。

```
brew cask install BetterZipQL
```

## QLStephen

QLStephen 可预览无拓展名的纯文本文件插件。

```
brew cask install QLStephen
```

## QuickLookCSV

预览 CSV 文件插件。

```
brew cask install quicklook-csv
```
